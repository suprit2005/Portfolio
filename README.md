# Portfolio Website

A full-stack Django portfolio application built with a structured backend architecture and modern UI.

## Tech Stack

- Python
- Django
- PostgreSQL
- Django REST Framework
- HTML / CSS / JavaScript

## Features

- Authentication system (Signup, Login, Logout)
- Protected dashboard access
- Admin-based content management
- Projects CRUD
- Skills with categories
- Education & Achievements management
- Contact form with email integration
- REST API endpoints
- Token-based authentication
- Dark/Light mode UI

## Project Structure

- accounts_app
- projects_app
- skills_app
- achievements_app
- contact_app
- api_app

## Setup Instructions

1. Clone repository
2. Create virtual environment
3. Install dependencies
4. Configure .env file
5. Run migrations
6. Create superuser
7. Start development server

## Environment Variables Required

- SECRET_KEY
- DEBUG
- DB_NAME
- DB_USER
- DB_PASSWORD
- DB_HOST
- DB_PORT
- EMAIL_HOST
- EMAIL_HOST_USER
- EMAIL_HOST_PASSWORD
